# Sheltr

Sheltr is a mobile application designed to simplify and modernize the pet adoption process. It provides users with an intuitive platform to browse available pets, mark favorites, communicate with adoption centers, and manage their adoption activity. The app is built for individuals or families looking to adopt pets and for organizations that want a simple digital system for listing their adoptable animals.
